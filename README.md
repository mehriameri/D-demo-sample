### `Getting Started`
Click the demo link or clone/download the repository on your local machine. \
Install dependencies\
npm install\
npm run server\
npm start
### `Built With:`
•	React js\
•	React Router\
•	React Hooks\
•	Redux\
•	Redux-Persist\
•	Json server as a fake API\
•	Tailwind Css
